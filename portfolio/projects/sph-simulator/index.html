Overview

A real-time 2D fluid simulation built inside Godot 4 using the Smoothed Particle Hydrodynamics (SPH) method. All physics passes run on the GPU via GLSL compute shaders, enabling thousands of particles at interactive frame rates.

Physics

Each particle carries density ρ, pressure P, and velocity v. The simulation runs three compute passes per frame:

  • Density pass — accumulates SPH kernel contributions from neighbours within radius h
  • Force pass — computes pressure gradient and viscosity forces
  • Integration pass — symplectic Euler, boundary collision response

GPU Architecture

  • Particle data stored in paired SSBOs (ping-pong), avoiding read-write hazards
  • Spatial hashing grid accelerates neighbour search from O(n²) to near O(n)
  • Rendering: instanced quads with per-particle colour mapped to speed magnitude
