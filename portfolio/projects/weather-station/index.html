Weather Station — Embedded System Lab

2025  ·  Chulalongkorn University  ·  Role: Embedded Firmware Programmer
STM32 ESP32 MQTT

Overview

An end-to-end embedded IoT system collecting environmental data (temperature, humidity, pressure, light intensity index) and visualizing it on a live web dashboard.

System Architecture

STM32Sensor acquisition over analog inputs and digital pins, forwards data to ESP32 via UART
ESP32-S3Wi-Fi bridge; publishes JSON payloads to MQTT broker (EMQX)
